# How it works ?

Axiom works thanks to the brand new version of its AXCore-V.10 engine, it easily solves all demade requests, The bot works with "modules" activable by commands (soon a synchronized dashboard) in order to deactivate / activate a kit of commands easily such as moderation, animation, levels, economy ect ...

# Commands

### Configuration

| Prefix | Command |  Args 1 | Args 2 |                 Action                 |
|:------:|:-------:|:-------:|:------:|:--------------------------------------:|
|    >   | set-mod | Boolean | *none* | Enable / Disable the moderation module |
|    >   | set-lvl | Boolean | *none* |  Enable / Disable the leveling module  |
|    >   | set-eco | Boolean | *none* |   Enable / Disable the economy module  |
|    >   | set-fun | Boolean | *none* |  Enable / Disable the animation module |
|    >   | set-wlc | Boolean | *none* |  Enable / Disable the welcoming module |

### Moderation

| Prefix | Command |   Args 1  |    Args 2    |       Time      |                    Action                    |
|:------:|:-------:|:---------:|:------------:|:---------------:|:--------------------------------------------:|
|    >   |   ban   |  @Member  | ***Reason*** | (1h / 1d / 1m ) |     Deny access to a member of the server    |
|    >   |   kick  |  @Member  | ***Reason*** |      *none*     |        Expels a member from the server       |
|    >   |   mute  |  @Member  | ***Reason*** | (1h / 1d / 1m ) |        Prohibits member from speaking        |
|    >   |   warn  |  @Member  | ***Reason*** |      *none*     |          Notify the member mentioned         |
|    >   | usermod |  @Member  | ***Reason*** |      *none*     |  Displays a interface to view all sanctions  |
|    >   |  unban  | Member ID | ***Reason*** |      *none*     | Restores access to the server for the member |
|    >   |  unmute |  @Member  | ***Reason*** |      *none*     |          Allow member to speak again         |
